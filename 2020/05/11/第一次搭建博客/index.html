遇到的些许问题

  1. next主题设置并不生效不停的采取$ hexo clean $ hexo g && hexo s $ hexo clean && hexo g && hexo d

  2. APlayer插件放置的位置不理想，也不生成自己想要的效果,详情参考pjax导致无法初始化aplayer音乐播放器，主要问题如下：

1.Aplayer主页插件和歌单里面的切换有干扰

2.Aplayer的Meting.js由于没有CDN可以引用，只能下载MetingJS 1.2.0源码，引用到本地的路径,来达到效果

3.博客主页没有预计的RSS效果实现

4.NexT主题版本升级迁移数据耗时良久

5.中英文符号的切换导致博客页面没有预计效果

以上这些问题对一个刚刚接触这方面的我来说，有问题只能使劲gang Google。好在一切都已经进入了正轨，接下来就开始享受拥有博客的快乐时光。感谢自己不抛弃不放弃的精神！
